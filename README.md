# Home Intrusion Sensor (ESP32 + Espressif RainMaker)

A cloud-connected home intrusion sensor built on **ESP32 (C)** integrated with **Espressif RainMaker** for monitoring and alerts.

## Features
- Real-time intrusion detection (ultrasonic sensor)
- Cloud status updates via RainMaker
- Remote arming/disarming (optional)
- Basic event logging (optional)

## Tech stack
- ESP32 (C)
- Espressif RainMaker
- Sensor integration 

## Hardware
- ESP32 board model: ESP32C6 devkit
- Sensor: SparkFun Electret Microphone Breakout
- Power:5v
